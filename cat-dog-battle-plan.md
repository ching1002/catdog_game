# 貓狗大戰：橫向卷軸射擊網頁遊戲規劃

## 1. 專案目標

製作一款可在手機橫向遊玩、也可在電腦瀏覽器遊玩的橫向卷軸射擊網頁遊戲。

玩家操控主角貓咪，在橫向捲動的戰場中發射光球，擊退一波又一波飛來的狗狗敵人。遊戲包含分數、Wave、升級配點、Boss、暫停、自動存檔與手機虛擬方向鍵。

## 2. 安裝與素材生成 Skill

已從 GitHub repo 安裝 agent-sprite-forge 相關技能：

- GitHub repo: https://github.com/0x0funky/agent-sprite-forge
- 精靈圖技能: `generate2dsprite`
- 地圖背景技能: `generate2dmap`

後續用途：

- 生成貓咪主角精靈圖
- 生成狗狗小怪精靈圖
- 生成狗狗 Boss 精靈圖
- 生成貓咪攻擊光球動畫
- 生成爆炸、衝擊波、受傷特效
- 生成橫版淡色背景圖與場景素材

注意：安裝新的 Codex skill 後，需要重新啟動 Codex 才能讓新 skill 生效。

## 3. 遊戲主題

主題名稱：貓狗大戰

核心風格：

- 可愛
- 明亮
- 有動作感
- 角色與敵人清楚易辨識
- 適合手機橫向短時間遊玩

世界觀：

貓咪守護自己的地盤，狗狗軍團從畫面右側一波波進攻。玩家需要操作貓咪閃避、射擊、升級，撐過越來越高難度的 Wave，並擊敗每 5 波出現的狗狗 Boss。

## 4. 核心玩法

遊戲類型：

- 2D 橫向卷軸射擊
- 自動射擊為主
- 玩家專注於走位、閃避、升級配點

基本流程：

1. 玩家操控貓咪在畫面左半部或全畫面範圍移動。
2. 狗狗敵人從畫面右側生成並往左移動。
3. 貓咪自動發射光球攻擊敵人。
4. 擊中敵人造成傷害。
5. 敵人血量歸零後爆炸，玩家獲得分數與經驗。
6. 玩家累積經驗升級，取得配點。
7. 每個 Wave 結束後難度提高。
8. 每 5 個 Wave 出現 Boss。
9. 貓咪體力歸零後遊戲結束。

## 5. 操作設計

### 手機操作

目標裝置：

- 手機瀏覽器
- 橫向螢幕
- 觸控操作

手機 UI 配置：

- 左下角：虛擬方向鍵
- 右下角：技能鍵、暫停鍵，若採手動射擊則放攻擊鍵
- 上方：血量、分數、Wave、等級、升級點數

方向鍵：

- 上
- 下
- 左
- 右

手機操作建議：

- 採用自動射擊
- 玩家只需要用左下角方向鍵控制貓咪移動
- 右下角保留技能鍵與暫停鍵
- 按鈕要夠大，避免誤觸
- UI 不遮擋角色、敵人與彈幕

### 電腦操作

鍵盤控制：

- `W`: 向上移動
- `A`: 向左移動
- `S`: 向下移動
- `D`: 向右移動
- `P`: 暫停或恢復
- `Space`: 技能或手動攻擊，視最終設計決定

滑鼠操作：

- 點擊升級選項
- 點擊暫停
- 點擊重新開始
- 點擊設定按鈕

## 6. 主角設計

主角：貓咪

視覺方向：

- 可愛貓咪
- 小披風或鈴鐺
- 表情有精神
- 輪廓清楚，手機小畫面也能辨識

必要動畫：

- 待機漂浮
- 向上移動
- 向下移動
- 向左移動
- 向右移動
- 發射光球
- 受傷閃爍
- 死亡或倒下

攻擊表現：

- 普通攻擊：發射小型發光毛球
- 升級攻擊：雙發毛球、散射毛球、穿透毛球、爆裂毛球
- 攻擊色彩：藍白、金黃、粉色可混搭，避免與敵方子彈混淆

## 7. 敵人設計

所有小怪與 Boss 都以狗為主體。

### 小型狗

定位：

- 最基礎敵人
- 數量多
- 血量低
- 速度中等

行為：

- 從右往左直線移動
- 撞到貓咪造成傷害

### 快速狗

定位：

- 速度型敵人
- 血量低
- 逼迫玩家走位

行為：

- 高速衝刺
- 可能有輕微上下擺動

### 大型狗

定位：

- 坦克型敵人
- 血量高
- 體型大

行為：

- 慢速前進
- 撞擊傷害較高

### 飛盤狗

定位：

- 遠程敵人

行為：

- 會發射飛盤子彈
- 飛盤可直線或弧線移動

### 骨頭狗

定位：

- 彈幕型小怪

行為：

- 發射骨頭子彈
- 隨 Wave 提升射擊頻率

## 8. Boss 設計

Boss 每 5 個 Wave 出現一次。

Boss 特色：

- 體型大
- 血量高
- 攻擊模式多
- 擊敗後給大量分數與經驗
- 出場時暫停普通敵人生成，或降低普通敵人生成量

### Boss 1：大型柴犬隊長

出現時間：

- Wave 5

攻擊方式：

- 直線骨頭彈
- 短距離衝撞
- 召喚小型狗

### Boss 2：狗屋堡壘

出現時間：

- Wave 10

攻擊方式：

- 多方向骨頭彈幕
- 從狗屋門口召喚小狗
- 慢速壓迫玩家空間

### Boss 3：機甲狗

出現時間：

- Wave 15

攻擊方式：

- 雷射骨頭砲
- 飛盤散射
- 追蹤彈

### 後續 Boss

可依照 Wave 循環強化：

- 提升血量
- 提升攻擊頻率
- 增加彈幕密度
- 增加召喚敵人數量

## 9. Wave 難度曲線

Wave 成長規則：

- Wave 越高，敵人數量越多
- Wave 越高，敵人速度越快
- Wave 越高，敵人血量小幅提升
- 每隔數個 Wave 解鎖新敵人類型
- 每 5 Wave 出現 Boss

建議節奏：

- Wave 1-3：教學與暖身，只有小型狗
- Wave 4：加入快速狗
- Wave 5：Boss 1
- Wave 6-8：加入大型狗
- Wave 9：加入飛盤狗
- Wave 10：Boss 2
- Wave 11-14：加入骨頭狗與更密集的敵人組合
- Wave 15：Boss 3

## 10. 升級與配點機制

經驗來源：

- 擊敗小型狗
- 擊敗大型狗
- 擊敗遠程狗
- 擊敗 Boss

升級規則：

- 累積經驗達門檻後升級
- 每升 1 級獲得 1 點配點
- 升級時可暫停戰鬥，讓玩家選擇能力

能力項目：

- 體力上限：增加最大 HP
- 火力：提升子彈傷害
- 射速：降低自動射擊間隔
- 防禦：降低受到的傷害
- 機動力：提升移動速度

平衡方向：

- 前期升級速度較快，讓玩家快速感覺變強
- 中期開始需要選擇流派
- 後期敵人彈幕變多，防禦與機動力價值提升
- 火力過高時也不能完全無腦通關，Boss 仍需要走位

## 11. 血量、傷害與死亡

貓咪狀態：

- 有總體力條
- 受到撞擊或子彈會扣血
- 受傷後短時間無敵閃爍
- 體力歸零後遊戲結束

敵人傷害：

- 小型狗撞擊：低傷害
- 快速狗撞擊：中傷害
- 大型狗撞擊：高傷害
- 飛盤：中傷害
- Boss 彈幕：中高傷害
- Boss 衝撞：高傷害

## 12. 分數與獎勵

分數來源：

- 擊敗敵人
- 擊敗 Boss
- 完成 Wave
- 連續擊破加成

獎勵設計：

- Boss 掉大量經驗
- 每 5 Wave 可給額外升級點或補血
- 高分可存成瀏覽器本機最高紀錄

## 13. 暫停與存檔

暫停功能：

- 手機：右下角暫停按鈕
- 電腦：`P` 鍵
- 暫停時停止敵人、子彈、背景捲動與碰撞判定

自動存檔：

- 使用瀏覽器 `localStorage`
- 不需要伺服器

建議保存內容：

- 最高分
- 最近分數
- 已達最高 Wave
- 玩家等級
- 升級配置
- 音量設定
- 操作設定

是否保存單局進度：

- 第一版建議只保存最高分與設定
- 進階版可保存當前局進度，但需要避免玩家重複讀檔刷分

## 14. 美術與場景

背景需求：

- 可愛橫版背景
- 顏色稍微刷淡
- 支援無縫橫向捲動
- 不干擾角色與子彈辨識

可用場景：

- 城市屋頂
- 公園草地
- 寵物店街道
- 貓咪秘密基地
- 狗狗軍團基地

前景與裝飾：

- 貓抓板
- 毛線球
- 狗屋
- 骨頭招牌
- 玩具球

視覺規則：

- 玩家子彈與敵方子彈顏色要明顯不同
- 背景透明度或飽和度降低
- 角色與敵人邊緣可加淡色描邊
- 爆炸特效要短暫明亮，但不能遮住彈幕太久

## 15. 音效與音樂

建議音效：

- 貓咪發射光球
- 擊中敵人
- 敵人爆炸
- 貓咪受傷
- 升級
- Boss 出場
- Boss 擊敗
- 暫停

音樂方向：

- 輕快
- 節奏明確
- 可愛但有戰鬥感

設定：

- 音樂開關
- 音效開關
- 音量調整

## 16. UI 規劃

遊戲中 UI：

- 左上：貓咪 HP 條
- 上方中央：Wave
- 右上：分數與等級
- 左下：手機方向鍵
- 右下：技能鍵與暫停鍵

升級 UI：

- 升級時顯示能力選擇面板
- 每個能力顯示目前等級
- 點擊後立即套用

遊戲結束 UI：

- 本局分數
- 最高分
- 到達 Wave
- 重新開始
- 返回主畫面

## 18. 建議開發階段

### 第一階段：可玩原型

目標：

- 貓咪可以移動
- 電腦支援 WASD
- 手機支援左下角方向鍵
- 貓咪可以自動射擊
- 狗狗敵人會生成與移動
- 有碰撞、扣血、分數、死亡

### 第二階段：橫向卷軸與美術

目標：

- 加入淡色橫版背景
- 背景可無縫捲動
- 加入基本貓咪與狗狗精靈圖
- 加入爆炸特效

### 第三階段：Wave 與升級

目標：

- 加入 Wave 系統
- 加入經驗值
- 加入升級配點
- 加入不同敵人類型

### 第四階段：Boss 與彈幕

目標：

- 每 5 Wave 出現 Boss
- Boss 有血量條
- Boss 會發射彈幕
- Boss 擊敗後給大量經驗與分數

### 第五階段：完整遊戲體驗

目標：

- 加入暫停
- 加入自動存檔
- 加入音效與音樂
- 加入主選單、遊戲結束畫面
- 調整手機橫向體驗
- 平衡難度曲線
