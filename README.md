# 貓狗大戰

橫向卷軸射擊網頁遊戲。手機請橫向遊玩，左下角方向鍵移動；電腦可用 `W`、`A`、`S`、`D` 操控。

## 遊玩方式

開啟 `index.html`，或使用 GitHub Pages 發佈後直接在瀏覽器遊玩。

流程：

1. 進入標題畫面。
2. 點擊「進入遊戲」。
3. 觀看前導影片，或按 `SKIP` 略過。
4. 操控貓咪擊退狗狗軍團。

## 操作

- 手機：左下角虛擬方向鍵
- 電腦：`WASD` 移動
- `P`：暫停
- `Space`：毛球爆發

## 技術

- HTML
- CSS
- JavaScript
- Canvas 2D
